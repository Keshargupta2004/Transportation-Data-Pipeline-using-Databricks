# Transportation-Data-Pipeline-using-Databricks
Built ETL pipeline using Medallion Architecture (Bronze → Silver → Gold)  Created dimension tables (dim_date) and analytics view (final_sales_view)  Processed and transformed data using PySpark and SQL  Automated workflows using Databricks Jobs on Microsoft Azure
